# 🐈 Cats

## Scripts


### `cats.py`

Asks you for cat names and saves them to `list.txt`

### `random_cat.py`

Randomly selects a cat name from `list.txt` to help you name your next feline friend

### `password.py`

Generates a secure password with customizable options, you can even include a random cat name from `list.txt.`

### `new-feature.sh`

Automates the Git feature branch workflow:

Creating a new branch from your commit message.
Staging all changes.
Creating a commit.
Pushing the branch to GitHub.
Preparing the repository for a Pull Request 🦈

### Makefile

Run common project tasks with:

```bash
make help
```

Available commands:

- `make help` – Show available commands.
- `make password` – Generate a secure password.
- `make random-cat` – Pick a random cat name.
- `make clean` – Placeholder for future cleanup tasks.
